# 🌷 Girlhood Spectrum

**Um cantinho da internet para os teus sentimentos**

Mood tracker para o projeto Technovation - Um espaço seguro e sem julgamentos para registar e acompanhar os teus sentimentos.

## ✨ Features

- 🎯 **Login Simples** - Apenas o teu nome, nada mais
- 😊 **5 Emoções para Escolher** - Feliz, Normal, Triste, Medo, Zangado
- 📊 **Intensidade do Sentimento** - Escala de 1-10 para cada mood
- ⏰ **Timestamps** - Regista quando e a que horas sentiste cada emoção
- 📈 **Histórico Pessoal** - Acompanha os teus sentimentos ao longo do tempo
- 📊 **Estatísticas** - Vê o teu mood mais frequente e intensidade média
- 👥 **Comunidade** - Vê como as outras pessoas se estão a sentir

## 🎮 Novo: Mood Tracking & Friendship Gamification

Uma inovação excitante que transforma o apoio emocional num jogo de amizade!

### Core Features de Gamificação:

- **👥 Gerenciamento de Amigas** - Adiciona amigas à tua rede de apoio
  - Reage aos moods delas (+5 pontos)
  - Deixa comentários de apoio (+10 pontos)
  - Acumula pontos por ações de empatia

- **🎯 Desafios de Bem-Estar Pessoal** - Completa pequenos desafios diários
  - Meditação (🧘 +25 pontos)
  - Caminhada (🚶 +30 pontos)
  - Beber água (💧 +20 pontos)
  - E muitos mais!
  - Ganha +50 pontos por 7 dias consecutivos

- **🏆 Pódio de Empatia** - Um leaderboard baseado em empatia, não em competição
  - Top 3 Amigas mais Apoiadoras showcaseadas com medalhas
  - Leaderboard completo de toda a comunidade
  - Pontos ganhos por:
    - Reações e comentários (+5-10 pontos)
    - Completar desafios pessoais (+20 pontos)
    - Consistência (7+ dias) (+50 pontos)
    - Apoiar amiga em momento difícil (+15 pontos)

### Proposta de Valor Única:

> "Transforma o apoio emocional num jogo de amizade: rastreia o teu mood, apoia as tuas amigas e sobe no pódio de empatia."

### Por que Funciona:

- 🤝 Incentiva genuína comunicação emocional
- 💪 Motivação positiva para autocuidado e bem-estar
- 👯 Fortalece amizades através de ações de apoio
- 🎯 Competição saudável baseada em empatia
- 💖 Sem pressão - participação completamente voluntária

## 🚀 Como Usar

1. **Login** → Introduz o teu nome
2. **Seleciona Mood + Intensidade** → Escolhe como te sentes e a intensidade (1-10)
3. **Guarda** → Os teus dados são guardados localmente
4. **Ver Histórico Pessoal** → Acompanha os teus sentimentos e estatísticas
5. **Comunidade** → Vê os moods de todas as outras pessoas
6. **Novo: Adiciona Amigas** → Invita amigas e ganha pontos por apoiar
7. **Novo: Completa Desafios** → Cuida de ti e ganha pontos
8. **Novo: Sobe no Pódio** → Vê o teu ranking de empatia

## 📁 Estrutura das Páginas

- **index.html** - Login inicial + Teaser de Gamificação
- **mood.html** - Seleção de mood com intensidade

- **finalmood.html** - Moods da comunidade

## 💾 Armazenamento

- Todos os dados são guardados no navegador (localStorage)
- Dados de moods pessoais, amigas, pontos e desafios
- Sem servidores, totalmente privado
- Dados persistem mesmo após fechar o navegador
