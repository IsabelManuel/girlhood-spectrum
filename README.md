# 🌷 Girlhood Spectrum

**Um cantinho da internet para os teus sentimentos**

Mood tracker para o projeto Technovation - Um espaço seguro e sem julgamentos para registar e acompanhar os teus sentimentos.

## ✨ Features

- 🎯 **Login Simples** - Apenas o teu nome, nada mais
- 😊 **5 Emoções para Escolher** - Feliz, Normal, Triste, Medo, Zangado
- 📊 **Intensidade do Sentimento** - Escala de 1-10 para cada mood
- ⏰ **Timestamps** - Regista quando e a que horas sentiste cada emoção
- 📈 **Histórico Pessoal** - Acompanha os teus sentimentos ao longo do tempo
- 📊 **Estatísticas** - Vê o teu mood mais frequente e intensidade média
- 👥 **Comunidade** - Vê como as outras pessoas se estão a sentir

## 🚀 Como Usar

1. **Login** → Introduz o teu nome
2. **Seleciona Mood + Intensidade** → Escolhe como te sentes e a intensidade (1-10)
3. **Guarda** → Os teus dados são guardados localmente
4. **Ver Histórico Pessoal** → Acompanha os teus sentimentos e estatísticas
5. **Comunidade** → Vê os moods de todas as outras pessoas

## 📁 Estrutura das Páginas

- **index.html** - Login inicial
- **mood.html** - Seleção de mood com intensidade
- **history.html** - Teu histórico pessoal e estatísticas
- **finalmood.html** - Moods da comunidade
- **script.js** - Lógica da aplicação com localStorage
- **styles.css** - Design com gradiente animado e glassmorphism

## 💾 Armazenamento

- Todos os dados são guardados no navegador (localStorage)
- Sem servidores, totalmente privado
- Dados persistem mesmo após fechar o navegador

## 🎨 Design

- Gradiente animado suave
- Glassmorphism cards
- Cores em tons de roxo e rosa
- Responsive e amigável para mobile
